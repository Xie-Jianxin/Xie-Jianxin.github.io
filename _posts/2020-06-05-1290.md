基本思路：

class Solution {

    public int getDecimalValue(ListNode head) {
	
        int decimal = 0;
		
        while(head != null){
		
            decimal = decimal * 2 + head.val;
			
            head = head.next;
			
        }
		
        return decimal;
		
    }
	
}

时间复杂度：O(n)

空间复杂度：O(1)
