基本思路：快慢指针法

`{``%` highlight language `%``}`
class Solution {
    public ListNode middleNode(ListNode head) {	
        ListNode low=head,fast=head;		
        if(head.next==null)	
            return head;		
        while(fast!=null&&fast.next!=null){	
            low=low.next;	
            fast=fast.next.next;	
        }
        return low;
    }
}
`{``%` endhighlight `%``}`

时间复杂度：O(n)

空间复杂度：O(1)
